# meshdb

<!-- badges: start -->

<!-- badges: end -->

The meshdb package provides a tidy interface to the MeSH ontology, which
is stored in local `*.parquet` files.

## Installation

You can install the development version of meshdb like so:

``` r
remotes::install_github("aratusbio/meshdb")
```
